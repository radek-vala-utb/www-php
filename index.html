Úvod do PHP

PHP výukové příklady

Základní sada příkladů pro výuku syntaxe, podmínek, polí, funkcí a práce s formulářem. Tato úvodní stránka je statická, aby šla zobrazit i na GitHub Pages.

Kde to půjde spustit

GitHub Pages umí zobrazit tento přehled, ale neumí vykonávat PHP. Plně funkční lekce spustíš v GitHub Codespaces nebo lokálně přes vestavěný PHP server příkazem php -S localhost:8000.

01. Základy syntaxe a proměnné

Proměnné, datové typy, přetypování a skládání textového výstupu.

$name = "Anna";
$year = 2004;
$age = date("Y") - $year;

02. Podmínky a formuláře

Jednoduché rozhodování pomocí if, elseif a else.

if ($age < 18) {
    $group = "junior";
} else {
    $group = "dospělý";
}

03. Pole a cykly

Rozdělení textu do pole a postupný výpis hodnot ve foreach.

$items = explode(",", $input);
foreach ($items as $item) {
    echo trim($item);
}

04. Funkce a řetězce

Vlastní funkce, parametry, návratová hodnota a úprava textu.

function makeTitle(string $text): string {
    return strtoupper(trim($text));
}

05. Asociativní pole a foreach

Práce s klíči, hodnotami a filtrováním jednoduché datové struktury.

$student = [
    "jméno" => "Eva",
    "body" => 92,
];

06. GET a POST validace

Základní validace formulářových dat a bezpečný výpis pomocí htmlspecialchars().

$safeName = htmlspecialchars($name, ENT_QUOTES, "UTF-8");

07. SQLite a databáze

Základní práce s databází přes PDO, vytvoření tabulky a výpis dat.

$pdo = new PDO("sqlite:students.sqlite");
$pdo->exec("CREATE TABLE IF NOT EXISTS students (...)");

08. Kompletní CRUD nad SQLite

Vytvoření, výpis, úprava i smazání záznamu v jedné jednoduché aplikaci.

$statement = $pdo->prepare("UPDATE students SET ... WHERE id = :id");
$statement->execute([...]);

09. Objektové PHP a mini ORM

Modely, třídy, vazby mezi tabulkami a ORM-style práce s daty bez frameworku.

$student = Student::find(1);
$student->name = "Anna";
$student->save();

Jak pokračovat

Pokud chceš PHP opravdu spouštět, otevři tento projekt v prostředí s podporou PHP serveru. V Codespaces nebo lokálně použij index.php jako hlavní rozcestník.
