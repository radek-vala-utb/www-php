Uvod do PHP

PHP vyukove priklady

Zakladni sada prikladu pro vyuku syntaxe, podminek, poli, funkci a prace s formularem. Tato uvodni stranka je staticka, aby sla zobrazit i na GitHub Pages.

Kde to pujde spustit

GitHub Pages umi zobrazit tento prehled, ale neumi vykonavat PHP. Plne funkcni lekce spustis v GitHub Codespaces nebo lokalne pres vestaveny PHP server prikazem php -S localhost:8000.

01. Zaklady syntaxe a promenne

Promenne, datove typy, pretypovani a skladani textoveho vystupu.

$name = "Anna";
$year = 2004;
$age = date("Y") - $year;

02. Podminky a formulare

Jednoduche rozhodovani pomoci if, elseif a else.

if ($age < 18) {
    $group = "junior";
} else {
    $group = "dospely";
}

03. Pole a cykly

Rozdeleni textu do pole a postupny vypis hodnot ve foreach.

$items = explode(",", $input);
foreach ($items as $item) {
    echo trim($item);
}

04. Funkce a retezce

Vlastni funkce, parametry, navratova hodnota a uprava textu.

function makeTitle(string $text): string {
    return strtoupper(trim($text));
}

05. Asociativni pole a foreach

Prace s klici, hodnotami a filtrovani jednoduche datove struktury.

$student = [
    "jmeno" => "Eva",
    "body" => 92,
];

06. GET a POST validace

Zakladni validace formularovych dat a bezpecny vypis pomoci htmlspecialchars().

$safeName = htmlspecialchars($name, ENT_QUOTES, "UTF-8");

Jak pokracovat

Pokud chces PHP opravdu spoustet, otevri tento projekt v prostredi s podporou PHP serveru. V Codespaces nebo lokalne pouzij index.php jako hlavni rozcestnik.
